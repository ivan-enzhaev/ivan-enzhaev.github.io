Building "Hello World!" in C via Qt Creator Сборка "Hello World!" на Си через Qt Creator

1. Project Structure 1. Структура проекта

Create a folder and set up your files exactly as in the console version: Создайте папку и подготовьте файлы так же, как в консольной версии:

CMakeLists.txt
cmake_minimum_required(VERSION 3.21)
project(hello-world-c)

set(CMAKE_C_STANDARD 11)
set(CMAKE_C_STANDARD_REQUIRED ON)

add_executable(app src/main.c)
src/main.c
#include <stdio.h>

int main() {
    printf("Hello World!\n");
    return 0;
}

2. Opening the Project 2. Открытие проекта

  1. Launch Qt Creator. Запустите Qt Creator.
  2. Go to File > Open File or Project... Выберите Файл > Открыть файл или проект...
  3. Select the CMakeLists.txt file in your folder. Выберите файл CMakeLists.txt в вашей папке.

3. Configure Project (Kits) 3. Настройка проекта (Kits)

In the "Configure Project" tab, ensure your MinGW 13.1 kit is selected. На вкладке "Configure Project" убедитесь, что выбран комплект (Kit) с MinGW 13.1.

Click the Configure Project button to generate the CMake cache. Нажмите кнопку Configure Project, чтобы сгенерировать кэш CMake.

4. Build and Run 4. Сборка и запуск

  • Press Ctrl + B to build the project. Нажмите Ctrl + B для сборки проекта.
  • Press Ctrl + R to run the application. Нажмите Ctrl + R для запуска приложения.

Ctrl + R

5. Viewing Output 5. Просмотр вывода

Check the Application Output (Вывод приложения) panel at the bottom of Qt Creator. You should see: Проверьте панель Вывод приложения в нижней части Qt Creator. Вы должны увидеть:

Hello World!
Tip / Совет
Qt Creator automatically handles the build directory (usually named build-hello-world...). You don't need manual .bat scripts when using this IDE! Qt Creator автоматически управляет папкой сборки. Вам не нужны .bat скрипты при использовании этой IDE!
