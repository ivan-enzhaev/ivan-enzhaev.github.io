C/C++ Development Environment Setup Настройка среды разработки на C/C++

1. MinGW 13.1 (WinLibs) 1. MinGW 13.1 (WinLibs)

Go to winlibs.com. Download the GCC 13.1.0 release (Win64, POSIX threads, SEH exceptions): Перейдите на winlibs.com. Скачайте версию GCC 13.1.0 (Win64, POSIX threads, SEH exceptions):

Installation: Extract the folder to a simple path without spaces, for example: C:\mingw64_13.1 Установка: Извлеките папку по простому пути без пробелов, например: C:\mingw64_13.1.

2. Path Configuration 2. Настройка переменных окружения

Add the MinGW bin folder to your System Path to use g++ and make in any terminal. Добавьте папку bin вашего MinGW в системную переменную Path, чтобы использовать g++ и make в любом терминале. 

# Add this to your System PATH:
C:\mingw64_13.1\bin

3. CMake Installation 3. Установка CMake

Download the Windows x64 Installer (.msi) from the Official CMake Website. Скачайте установщик Windows x64 (.msi) с официального сайта CMake.

During installation, select: "Add CMake to the system PATH for all users". Во время установки выберите: "Add CMake to the system PATH for all users".

4. IDE & Editor Setup (Sublime, Qt Creator, CLion) 4. Установка IDE и редакторов (Sublime, Qt Creator, CLion)

  • Sublime Text 4: Download from the official website. It is a sophisticated text editor for code, markup, and prose that works perfectly with MinGW via the build systems. Скачайте с официального сайта. Это продвинутый текстовый редактор для кода, разметки и прозы, который отлично работает с MinGW через системы сборки.
  • Qt Creator: Download the standalone Windows installer (.msi) from the Official GitHub Releases. This provides the IDE without requiring a full Qt Framework installation. Скачайте автономный установщик Windows (.msi) с официальных релизов GitHub. Это позволит установить IDE без необходимости загрузки всего фреймворка Qt.
  • CLion: Install via JetBrains. CLion is now free for non-commercial use (learning, hobby development, and open-source) Установите через JetBrains. CLion теперь бесплатна для некоммерческого использования (обучение, хобби, open-source)
Verification: Open CMD and type g++ --version and cmake --version to ensure everything is working! Проверка: Откройте CMD и введите g++ --version и cmake --version, чтобы убедиться, что всё работает!
