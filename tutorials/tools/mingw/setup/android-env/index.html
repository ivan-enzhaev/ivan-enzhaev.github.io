Android Environment Setup Установка окружения Android

1. Android SDK Command-Line Tools 1. Android SDK Command-Line Tools

Go to the Android Studio download page. Scroll to the bottom and download the commandlinetools-win-xxxxxx_latest.zip archive. Перейдите по ссылке Android Studio и в самом низу страницы скачайте архив commandlinetools-win-xxxxxx_latest.zip.

Important: Create a folder named AndroidSDK (e.g., on C:\). Extract the archive so your path looks like this: Важно: Создайте папку AndroidSDK (например, на диске C:\). Извлеките архив так, чтобы путь выглядел следующим образом:
C:\AndroidSDK\cmdline-tools\latest\

Note: Inside cmdline-tools, you must create a folder named latest and move the original bin, lib, source.properties, and notice.txt into it. Примечание: Внутри папки cmdline-tools необходимо создать папку latest и переместить в неё содержимое (bin, lib и т.д.).

2. Licenses and Component Installation 2. Лицензии и установка компонентов

Open CMD in C:\AndroidSDK\cmdline-tools\latest\bin and run the following to accept licenses (press Y + Enter for all): Откройте CMD в папке C:\AndroidSDK\cmdline-tools\latest\bin и выполните команду для принятия лицензий (нажимайте Y + Enter): 

sdkmanager --licenses

Now install the necessary platforms, build tools, and NDK: Теперь установите необходимые платформы, build-tools и NDK: 

sdkmanager "platform-tools" "build-tools;35.0.0" "platforms;android-35" "ndk;27.0.12077973"

3. Environment Variables (PATH) 3. Переменные среды (PATH)

Add the following variables to your System Environment Variables: Добавьте следующие переменные в системные переменные среды: 

# Variables
ANDROID_HOME: C:\AndroidSDK
ANDROID_NDK_HOME: C:\AndroidSDK\ndk\27.0.12077973;

# Add to Path
%ANDROID_HOME%\cmdline-tools\latest\bin
%ANDROID_HOME%\platform-tools
%ANDROID_HOME%\build-tools\35.0.0

4. JDK and CMake 4. JDK и CMake

  • Download and install JDK 17. Скачайте и установите JDK 17.
  • Download and install CMake (look for cmake-x.x.x-windows-x86_64.msi). Скачайте и установите CMake (ищите cmake-x.x.x-windows-x86_64.msi).
